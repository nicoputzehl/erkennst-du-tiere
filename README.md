# Erkennst du: Tiere?

Ist ein Quiz bei dem man Tiere erraten muss.
Es wird ein Bild von einem Tier gezeigt, und man muss den Namen es Tieres eingeben.

## Ideen

* Logging mit Farben
* komplexere Unlock Bedingungen
* Hint System
* Algoryhtmen um Rechtschreibung auszugleichen
